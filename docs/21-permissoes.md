## Componentes

- `PermissionGroup`
- `Permission`
- `RolePermission`
- `IPermissionService`

## Roles previstas

- Administrator
- Manager
- Professional
- Collaborator

As permissões devem ser usadas para autorização fina quando roles forem amplas demais.
